# Downloads

Download the latest version of Nodus from the official website:

**[Download Nodus](https://sorenwacker.net/nodus/)**

The download page includes installers for macOS, Windows, and Linux, along with installation instructions.

## System Requirements

| Platform | Requirements |
|----------|--------------|
| **macOS** | macOS 10.15 (Catalina) or later |
| **Windows** | Windows 10 (64-bit) or later |
| **Linux** | Ubuntu 20.04+, Debian 11+, Fedora 36+ |

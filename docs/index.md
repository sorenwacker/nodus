# Nodus

**Local-first knowledge graph with EU sovereignty.**

Nodus is a canvas-based visual thinking tool where research nodes, Typst math, and Obsidian vaults live on a single workspace. Your data stays on your device.

[Download :material-download:](https://github.com/sorenwacker/nodus/releases){ .md-button .md-button--primary }
[View Source :material-github:](https://github.com/sorenwacker/nodus){ .md-button }

---

## Key Features

<div class="grid cards" markdown>

- :material-brush: **Single Canvas**
  Document and whiteboard in one place

- :material-math-integral: **Native Typst**
  Sub-second math rendering

- :material-sync: **Obsidian Bridge**
  Bi-directional vault sync

- :material-database: **Local-First**
  Your data never leaves your device

- :material-robot: **LLM Integration**
  Connect local or cloud AI

- :material-export: **Export Options**
  PDF, Typst, and Markdown

</div>

## Quick Start

1. Download the latest release for your platform
2. Install and launch Nodus
3. Double-click on the canvas to create your first node
4. Connect nodes by dragging between them

See [Getting Started](getting-started.md) for detailed instructions.
